%STARTSECTION{type="templateonly"}%
This is an template for an example document that demonstrates some of
the features of the %SYSTEMWEB%.WorkflowPlugin. Documents based on this
template will be controlled by the %WORKFLOW% workflow.

The table below is part of the example content, and is generated by expanding
some of the macros described in %SYSTEMWEB%.VarWORKFLOW

You can use the button on the WebHome page to create a new topic
you can experiment with.

<!--
   * Set DENYTOPICCHANGE = *
   * Set WORKFLOWTRANSITION = DISABLED
-->
%ENDSECTION{type="templateonly"}%

| *Macro* | *Description* | *Expands to*|
| =%<nop>WORKFLOWSTATE%= | Current state of the document | %WORKFLOWSTATE% |
| =%<nop>WORKFLOWSTATEMESSAGE%= | State message | %WORKFLOWSTATEMESSAGE% |
| =%<nop>WORKFLOWTRANSITIONS%= | Transitions available to you | %WORKFLOWTRANSITION% |
| =%<nop>WORKFLOWLAST{"APPROVED" format="$date by $author"}%= | Last time in APPROVED state | %WORKFLOWLAST{"APPROVED" format="$date by $author"}% |
| =%<nop>WORKFLOWLASTVERSION{"APPROVED"}%= | Link to version when it was last in APPROVED state | %WORKFLOWLASTVERSION{"APPROVED"}% |
| =%<nop>WORKFLOWHISTORY%= | Workflow history | %WORKFLOWHISTORY% |
| =%<nop>WORKFLOWFORK{newnames="ControlledDocumentAUTOINC000" log="off"}%= | Fork the document to a new topic | %WORKFLOWFORK{newnames="ControlledDocumentAUTOINC000"}% |

Other =WORKFLOW*= preferences defined in %WORKFLOW% are defined
here too:
| =WORKFLOWNOTICE= | %WORKFLOWNOTICE% |
| =WORKFLOWSET= | %WORKFLOWSET% |
| =WORKFLOWDEBUG= | %WORKFLOWDEBUG% |
